# Integrations

## Import Databases

[Import your existing databases using a connection string](#)

Appmint Vibe Studio supports seamless integration with existing PostgreSQL databases. By providing a connection string,
 Appmint Vibe Studio automatically reads your schema and generates full-stack applications that leverage your existing data models. {{ className: 'lead' }}

### Features

- [**One-Click Import**:](#) Connect to any PostgreSQL-compatible database using a connection string. {{ className: 'lead' }}
- [**Automatic Schema Reading**:](#) Appmint Vibe Studio analyzes your database schema, including tables, relationships, and constraints, to understand your data model. {{ className: 'lead' }}
- [**Contextual App Generation**:](#) Applications generated by Appmint Vibe Studio are aware of your data structure and use it intelligently throughout the backend and frontend. {{ className: 'lead' }}
- [**Schema Sync**:](#) Use the Pull database schema button in the Infrastructure tab to sync your schema changes at any time. {{ className: 'lead' }}
- [**Environment Portability**:](#) Applications continue to function seamlessly across environments when deployed with Appmint Vibe Studio Cloud. {{ className: 'lead' }}

## Use Cases

- [**Dashboards**](#) Build interactive dashboards to visualize, explore, and manage your data in real time. {{ className: 'lead' }}
- [**CRM Extensions**](#) Integrate with user databases to trigger actions, automate workflows, and enrich customer data. {{ className: 'lead' }}
- [**Custom Agents**](#) Combine your database with an AI API key to create agents that automate internal processes and decision-making. {{ className: 'lead' }}
- [**Internal Tools**](#) Streamline operations with tools for user lookups, permission management, approval flows, and more. {{ className: 'lead' }}

## Step-by-step guide

1. **Connect your database** {{ className: 'lead' }}
    
    Provide your PostgreSQL connection string to import your existing database. 
    You can do this either in the prompt area when creating a new app, or in the Infrastructure view within an existing app, 
    by clicking Connect database in the External databases section. {{ className: 'lead' }}
  
   - connectdbprompt.png {{ className: 'lead' }}
    
   - connectdbinfra.png {{ className: 'lead' }}
    
2. **Appmint Vibe Studio automatically analyzes your schema** {{ className: 'lead' }}
    
    Appmint Vibe Studio automatically reads and understands your schema, adding this to the context of your application. You don’t need to do anything. {{ className: 'lead' }}
    
3. **Prompt to build your app** {{ className: 'lead' }}
    
    Prompt Appmint Vibe Studio to build an application, it will automatically know how to use your database and what the schema looks like. {{ className: 'lead' }}
    
4. **Pull in updates** {{ className: 'lead' }}
    
    Your database schema can continue evolving outside of Appmint Vibe Studio using your preferred tools and workflows. If changes occur,
     click Pull database schema in the Infrastructure viewer to update Appmint Vibe Studio’s understanding of your schema automatically. {{ className: 'lead' }}
