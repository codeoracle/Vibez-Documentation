# Deployment

Deploy your Appmint Vibe Studio applications to Appmint Vibe Studio Cloud or your own AWS/GCP infrastructure {{ className: 'lead' }}

### When you’re ready to share your app with the world, it’s time to deploy it.

Appmint Vibe Studio apps can be deployed automatically using Appmint Vibe Studio’s integration with Appmint Vibe Studio Cloud. 
Just click Deploy in the top-right corner of Appmint Vibe Studio to open the deployment dialog. From there, you 
can choose to deploy to a free Appmint Vibe Studio Cloud-hosted environment or connect your own AWS or GCP account. {{ className: 'lead' }}

 ## Deploy button

[Deploy Modal](#)

Appmint Vibe Studio Cloud’s built-in hosting is ideal for small-scale projects and comes with usage limits.
 To remove these limits, you can deploy to your own AWS or GCP account, where resources are billed directly to you. {{ className: 'lead' }}

Same experience either way: Whether you deploy to Appmint Vibe Studio’s infrastructure or your own cloud, you 
get the same monitoring, scaling, and developer experience through Appmint Vibe Studio Cloud. {{ className: 'lead' }}

## Deploying to your own cloud

With Appmint Vibe Studio Cloud you can automate deployments to your own AWS or GCP account. This requires an Appmint
 Vibe Studio Cloud Pro subscription, which enables Appmint Vibe Studio Cloud to orchestrate the deployment to your cloud provider. {{ className: 'lead' }}

[Here’s what you get:](#)

- One click deploys - Appmint Vibe Studio Cloud handles CI/CD and orchestrates infrastructure provisioning {{ className: 'lead' }}
- Data sovereignty - Your data stays in your cloud account {{ className: 'lead' }}
- Compliance - Meet enterprise and regulatory requirements {{ className: 'lead' }}
- Cost control - Use existing cloud discounts and reserved instances {{ className: 'lead' }}
- Security - Deploy within your existing VPC and security models {{ className: 'lead' }}
