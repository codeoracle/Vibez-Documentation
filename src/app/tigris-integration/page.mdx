# Tigris Integration

## S3-compatible object storage with global edge distribution for files, images, and media

Appmint Vibe Studio makes object storage simple with Tigris. Get S3-compatible storage with global edge distribution for your files and media. {{ className: 'lead' }}

## How it works

- Describe your storage needs in a prompt {{ className: 'lead' }}
- Appmint Vibe Studio generates Tigris integration with bucket management {{ className: 'lead' }}
- Add your credentials to secrets when prompted {{ className: 'lead' }}
- LLM results vary, but Tigris integrations typically work in one shot since it uses the standard S3 API. {{ className: 'lead' }}

## Example prompt

```
- Add file storage using Tigris for user profile images with automatic resizing,
- document uploads with virus scanning, and video storage with CDN delivery
- and signed URLs for private content

```
